Description:

Image name and light positions are stored in JSON files.
In add mode, type in the text field to define the light name.
Rollover lights to see their data/properties.

Controls:

View mode:
LMB - drag to pan
Mouse Scroll - zoom
RMB - rotate

Add mode:
LMB - add light
RMB - remove light

S - save and download data
D - load c1basement1
F - load c1basement2
C - reset camera
Space - toggle add/view mode
